## Project Overview

This project is a **rule-based chatbot** developed using Python. It simulates a conversation between a student and a college library assistant. The chatbot provides basic library services such as book search, issuing rules, return process, and fine calculation.

---

## Objectives

* To understand the concept of rule-based chatbots
* To implement **if-else conditions** for decision making
* To perform **mathematical calculations** (library fine)
* To handle errors using **exception handling**
* To simulate real-world interaction in a library system

---

## Features

* Greeting and user interaction
* Book search functionality
* Book issue limit checking
* Book return system
* Fine calculation for late returns
* Error handling for invalid inputs

---

## Technologies Used

* Python
* Jupyter Notebook / Google Colab

---

## How It Works

The chatbot works based on predefined rules using **if-else statements**.
It takes user input, processes the request, and provides appropriate responses.

---

## How to Run

1. Open the notebook in Google Colab or Jupyter Notebook
2. Run all cells
3. Interact with the chatbot via input

---

## Sample Interaction

```
You: hello  
Bot: Hello! How can I assist you?  

You: search book  
Enter book name: python  
Bot: 'Python' is available  

You: fine  
Enter days: 3

```

## Conclusion

This project demonstrates how basic Python concepts like conditions, loops, and exception handling can be used to build an interactive chatbot for real-world applications.
